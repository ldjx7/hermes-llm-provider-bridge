# Code of Conduct

This project aims to be practical, technical, and respectful.

Participants are expected to:

- Use welcoming and professional language.
- Focus critique on code, behavior, and design tradeoffs.
- Respect different experience levels and environments.
- Avoid harassment, personal attacks, or discriminatory conduct.

Unacceptable behavior includes:

- Insults, threats, or sustained disruption.
- Publishing private information without permission.
- Sexualized language or attention.
- Discrimination based on identity, background, or ability.

Project maintainers may remove comments, close issues, or restrict participation when behavior harms the project community.

To report a problem, use GitHub's private vulnerability reporting or contact the maintainers through the repository issue tracker if the matter is not sensitive.
